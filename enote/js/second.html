# 关于数组的常用方法整理

# 常用数组方法

# array.push() 在数组最后一项添加元素 会改变原数组 返回数组的新长度

# array.pop() 删除数组的最后一项 改变原数组 返回被删除的哪一个元素

# array.unshift() 在数组第一项前追加元素 改变原数组 返回数组的新长度

# array.shift() 删除数组的第一项元素 改变原素组 返回被删除的那个元素

# array.splice(index,length,'')

当只有两个参数的时候 表示从数组的第index开始删除length哥元素 当有三个参数的时候表示删除完后用str 替代

# Array.from

+ 将一个类数组对象或者可便利的对象转换成一个真正的数组
    + 所谓类数组对象最基本的要求就是具有length属性
    ```
    let arraylike = {
        0:'tom',
        1:'65',
        2: '男',
        3:['nike','adidas','puma'],
        'length': 4
    }
    let arr =Array.from(arraylike)
    console.log(arr)  ==> ['tom','65', '男',['nike','adidas','puma']]
    // 如果将上面代码中length属性去掉呢？实践证明，答案会是一个长度为0的空数组
    ```
    + 将字符串转换为数组
    ```
    let str = 'hello world'
    console.log(Array.from(str)) // ['h','e','l','l','o',' ','w','o','r','l','d']
    ```

# 数组的映射

# Array.map()方法 创建一个新的映射数组，而不是改变原数组

```
let arr = [1,3,5]
let numberarr = arr.map(item=>{
    return item+1
})
console.log(numberarr)  => [2,4,6]
```

# 数据的简化

# Array.reduce() 方法 通过调用callback函数来讲数组简化为一个值

```
let arr = [2,3,5]
function add (accumulator,number){
    return accumulator + number
}
const number = arr.reduce(add,1)
console.log(number)  // 11
// 将accumulator初始化为1，然后对每个累加数字和的数组项掉用add函数
```
